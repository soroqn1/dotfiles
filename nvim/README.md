# 💤 LazyVim

A starter template for [LazyVim](https://github.com/LazyVim/LazyVim).
Refer to the [documentation](https://lazyvim.github.io/installation) to get started.

## Custom Keymaps

The following keybindings have been added:

### General & Editor
- `Ctrl + p` — Search files by name (Telescope)
- `Ctrl + f` — Search text in the current file (Telescope)
- `Ctrl + g` — Search text across the whole project (Telescope)
- `Ctrl + a` — Select all text in the current file
- `Space + w` — Close the current file tab/buffer

### Neo-tree (File Explorer)
- `n` — Create a new file or folder (use a trailing slash `/` for folders, e.g. `src/`)
- `Y` (Shift + y) — Copy file path (Absolute/Relative) or filename to clipboard
